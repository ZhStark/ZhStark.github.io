大一宏晶杯总结

发表于 2017-03-09   |   分类于 技术   |     |   阅读次数

前言：

单片机（MPU）是本科阶段参加科技创新比赛极其重要的一块，用过单片机及其外设可以实现很多功能，比如大学生机器人大赛的主控芯片就是单片机，现在网络开源有很多基于STM32主控芯片的四轴飞行器，STM32就是一款32位的ARM内核单片机。

比赛简介：

该比赛是面向全校全体本科生的一个比赛，竞赛课题由相关专家组和企业拟定，比赛用单片机采用由宏晶科技有限公司统一提供的IAP15W4K58S4系列。（一款基于8051内核的8位单片机）

所做题目：

1、基于IAP15W4K58S4单片机的信号发生器：

基本要求：要求能产生方波，正弦波，三角波（电源自行制作）。
扩展功能：1.信号幅值，频率具体值的设定和更改（可选择利用无线进行设定，幅值范围-5V~5V，频率范围1KHz~10KHz）。 2.实时波形曲线绘制和显示。 3.三相电或多路信号波形显示。

  （该题目差不多是所有题目中最简单的一个，其他三个为：
2、基于IAP15W4K58S4单片机的十字路口车流量检测器：
基本要求：检测四个方向的车流量，并能进行数据储存，时间精确到秒
扩展功能：1.GPS定位，检测具体路口的车流量。2.无线数据传输至手机或者上位计算机。3.           
  上位机数据实时显示。
3、基于IAP15W4K58S4单片机的自平衡两轮车：
基本要求：实现平衡车的稳定直立，并按照场地设定好的路线进行循迹前进（使用电磁检测
方式），按照完成总路程用时长短进行排名。
4、    基于IAP15W4K58S4单片机的智能型计步器：
基本要求：1.计步器佩戴在身上能够记录并显示当前总步数。2.当佩戴者摔倒时，计步器能实
现报警功能。扩展功能：1.对计步器的GPS定位。 2.报警方式为向设定好的号码（可进行更
改）发送手机短信，并向该号码打电话进行报警。）

因为单片机我的专业相关程度很大，简单地控制都是通过单片机来完成，大二还有单片机课程，下学期学生会的事情少了一些，所以我想通过参加比赛来强迫自己去学习科创（科技创新）知识（虽然我不觉得捣鼓这玩意就叫做科创了，但大家都这么说，显得很高大上的样子，毕竟这是做科创的基础。），也是弥补上学期在这方面毫无进展的遗憾。拿个奖就更好了。

一个队伍要3个人，最开始拉舍友，他们都嫌难不愿参赛，找学长，学长已经有队伍了，最后拉了老乡妹纸进来，她又拉个她舍友，队伍算是齐了。

因为第一个不是辣么难，要是抽到第三个题目估计就直接放弃了。拿到题目后开始准备。每天学到1点睡觉，7点起床，以为一天六个半小时够了（午觉30min）。刚开始一周还可以，过了一周后身体有点撑不住了，每天都是同学拍拍肩膀告诉我下课了，精神很萎靡。在加上学习过程中遇到一些挫折有些精神崩溃。最后翘了几节课用来补觉，精神有些恢复。【有媒体采访山大泰山学堂的学生，他们说每天睡6个小时足够了，这可能与基因有关吧，让我想起高中那位一天睡6小时一点不困的同学，我这种嗜睡的体质证明我有些高度达不到，不过如果能达到上限也够了】第一个星期里把主要部分的内容学的差不多了，虽然中间遇到一个很蛋疼的事情：捣鼓了两天的DA模块始终不对，最后发现原来是DA模块是坏的。那几天精神挺崩的，不过也意识到自己的浮躁，【年轻人，不要浮躁】在遇到问题的时候没有按照一定顺序来检查，而是像钻进了牛角尖一直认为是自己的电路连错了。
第一周过后进度慢了下来，【查克拉在第一周用光了啊】一共有四周时间，第二周学了一下剩下的元器件用法，打算用仿真软件模拟，结果没有任何反应，应该是不太会用，放弃。（其实是让一队友负责仿真的，但不那么靠谱，有自己尝试，失败而终）第三周买的元器件到了，写了程序，用面包板连了电路，最开始是用蜂鸣器来间接表示发生的波形的不同（我真机智）发现了程序的问题，一点点调试。到了周末去电路实验室（只在周末开门的说）调试，结果示波器连个💩都没有。精神挺崩溃的，平静下来后再从前往后一点点找问题，到第四周周末用面包板调试成功了，开始焊板子。焊了一天，在周日下午5点焊好了，打算去实验室做最后调整的时候得知实验室停电关门了，心中顿时炸了，千万只羊驼在狂奔！山穷水尽之时听说工训有示波器，跑到工训去找，找到的时候人家已经关门了，工训只在白天开门，日了🐶了。夜里我机智的把波形周期调很长，用万用表测电压，通过示数判断输出波形是否可以。只是初测差不多。第五周周六上午比赛，只好翘了一节英语去工训借示波器，发现一些小问题但还好知道如何调整。并把频率调到1kHZ。周六去答辩，到我这个队的时候显示的乱七八糟（当时是以为我的发生器的问题，并不知道是示波器的问题，后来发现示波器的探针的线断了，估计是我前边的那一队撤线的时候拉断了），直接吓尿了，后来发现不是我的问题，换了示波器，成功显示，通过答辩。【我也是真够背的】

因为只做了基础功能，以为也就是三等奖了，毕竟参赛的好多大二大三的。没想到能拿个二等奖，老师真好~

后记：
这次比赛队友提供了一些资料和精神支持，东西基本是我一个人做出来的，不过如果没有他们的支持我真不确定在精神要崩的时候能不能坚持下来。

参加这次比赛也让我发现了自身的很多问题，最大的就是浮躁，不够淡定，这也是我会精神崩溃的原因。快速的信息时代带来的负面影响。看到网上一句话，铭记在心：年轻人，不要浮躁。

距离再远，路途再长，也要一步一步来。

那些熬夜通宵看起来好像很腻害，但对我来说，从稍长远看，得不偿失。我的身体不支持长时间熬夜。

要学的东西还有很多。

编程，学习实用的科学知识才是我的女朋友。

从答辩现场来看，做这玩意的美女还是不少的。
